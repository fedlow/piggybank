# Копилка — консольное приложение для управления целями накоплений

## Возможности
- Добавление/удаление/редактирование целей.
- Пополнение или уменьшение баланса с защитой от превышения цели.
- Автоматическое изменение статуса ("выполнена").
- Категоризация целей (Работа, Здоровье, Отдых и т.д.).
- Подсчёт общего прогресса по всем целям.
- Уведомления о достижении 10%, 25%, 50%, 75%, 90%, 100% цели.
- **Напоминания о дедлайнах** (если установлена дата завершения).
- **Прогноз даты завершения** на основе плановых еженедельных/ежемесячных пополнений.
- Сохранение всех данных в JSON-файл (`piggybank.json`).

## Запуск
1. Убедитесь, что установлен Python 3.6+.
2. Скачайте файл `piggybank.py`.
3. Запустите в терминале:
   ```bash
   python piggybank.py
